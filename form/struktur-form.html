Suatu halaman web dapat mengumpulkan informasi dari client ke server melalui suatu formulir. Pembuatan formulir pada dokumen HTML dilakukan dengan menggunakan elemen form. Pada elemen form terdapat dua attribute yang umumnya disertakan, yaitu action dan method.

attribute action dapat diisi dengan alamat penerima informasi (URL server) ketika formulir tersebut dikirimkan (submit).

Sedangkan attribute method diisi dengan salah satu cara pengiriman formulir, yaitu get dan post.

Method get memungkinkan pengiriman formulir ke server melalui penambahan data di akhir URL. Method post memungkinkan pengiriman formulir ke server melalui HTTP headers, sehingga tidak terlihat pada ur.l



Di dalam konten elemen form nantinya dapat diisi dengan berbagai jenis form controls.
