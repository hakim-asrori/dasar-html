Definition List digunakan untuk menampilkan istilah (term) dan definisinya (definition). Di dalam elemen definition list umumnya terdapat pasangan elemen definition term dan elemen definition. Misalnya sebagai berikut:
HyperText Markup Language (HTML)
Suatu standar bahasa untuk membentuk suatu struktur halaman web yang akan ditampilkan di browser
Cascading Style Sheet (CSS)
Suatu mekanisme sederhana untuk memberikan modifikasi terhadap halaman web, misalnya tipe huruf, warna, spasi, dan tata letak.
JavaScript (JS)
Suatu core (inti) bahasa pemrograman yang digunakan untuk mengembangkan teknologi web secara ringan, dinamis, serta fleksible baik pada client-side maupun server-side.
