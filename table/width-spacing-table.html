Untuk mengatur lebar (width) suatu kolom dan spasi antar cell dengan satuan pixel, baik di dalam cell (cellpadding) maupun di luar cell (cellspacing), dianjurkan agar menggunakan CSS. Penggunaan atribut width, cellpadding, cellspacing masih dapat digunakan walaupun di web modern sudah tidak dianjurkan lagi karena sudah outdate atau telah usang
