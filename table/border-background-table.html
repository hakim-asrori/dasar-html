Pada bagian sebelumnya, seringkali ditemukan atribut border pada elemen table. Hal ini dilakukan untuk menampilkan garis tepi pada elemen table dan td. Selain border, terdapat juga elemen bgcolor untuk memodifikasi table melalui perubahan warna suatu table.

Namun, baik elemen border maupun bgcolor, keduanya dianjurkan untuk tidak digunakan lagi pada web modern, karena semua pengelolaan tampilan dilakukan oleh pada CSS.
