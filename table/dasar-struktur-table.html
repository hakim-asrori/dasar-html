Elemen table memungkinkan penyajian informasi ke dalam format grid yang memudahkan terhadap pemahaman data yang kompleks dengan dua axis. Konten pada elemen table ditulis baris per baris.

Di dalam elemen table umumnya terdapat elemen tr dan elemen td. Elemen tr adalah singkatan dari table row, sedangkan td adalah singkatan dari table data.
